Page builder todo
--------------------

This page helps us to design pages on the go. It offers 3 types of page stucturing

1. 3 column view 
	This view provides 3 columns which consists of 2 sidebars (left and right) and main content  in the middle
2. 2 column view with left sidebar
	This view provides 2 column which consists of left side bar and main content on the right
3. 2 column view with right sidebar
	This view provides 2 column which consists of right side bar and main content on the left

Page builder can make use of forms which is already created and we can specify its position on the page ie., in sidebar or in main content
